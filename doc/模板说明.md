## 模板说明
### 添加片段
```markdown
    ## Snippet

    ```[Language]
    <代码片段内容>
    ```

    ## MetaData

    - Name
        - <代码片段名称>
    - Language
        - <代码片段所用语言>
```

### 编辑片段
```markdown
    ## Snippet
    
    ```[Language]
    <代码片段内容>
    ```
        
    ## MetaData
    
    - Tags
      - <代码片段标签>
      - <可以为多个>
      - <初次添加本片段时无需手动填写，由配置的LLM自动生成>
    - Description
      - <代码片段描述内容，默认也由LLM自动生成>
```
